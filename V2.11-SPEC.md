## New Drift Metrics (v2.11)
- **verificationEvidenceScore** (0.30): rounds with real verify after modify/ambiguous work
- **progressEventScore** (0.30): discrete meaningful progress (replaces closureRatio)
- **depthScore** (0.20): depth-like calls / real tool calls (kept, includes browser reads)
- **stabilityScore** (0.20): (stableStateRounds - oscillationCount) / validRounds (replaces transitionDensity)
